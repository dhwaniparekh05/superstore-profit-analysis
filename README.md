# superstore-profit-analysis

**Author:** Dhwani Parekh

## Overview
End-to-end data analytics project on 9,994 retail orders (2014–2017).
Uncovered $17K in losses, identified discount abuse, and segmented 793 customers using machine learning.

## Key Findings
- Orders with >20% discount lose an average of $156 per order
- Tables sub-category lost $17,725 over 4 years despite $206K in sales
- West region leads with 14.9% profit margin vs Central's 7.9%
- 111 customers identified as churned/at-risk via K-Means clustering
